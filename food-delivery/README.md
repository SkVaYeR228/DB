# Food Delivery System

Курсова робота з дисципліни "Бази даних".
Backend-сервіс для системи доставки їжі.

## Автор
Студент: Комар Тимофій ІМ-43

## Технології
- **Node.js** (середовище виконання)
- **PostgreSQL** (база даних)
- **Prisma ORM** (робота з даними)
- **Docker & Docker Compose** (контейнеризація)
- **Jest** (тестування)

## Як запустити проєкт

### 1. Запуск бази даних
```bash
docker-compose up -d
```
### 2. Налаштування схеми та даних
```bash
# Створення таблиць
npx prisma migrate dev --name init

# Заповнення тестовими данними
npx prisma db seed
```
### 3. Запуск тестів
```bash
npm test
```
### 4. Запуск сервера
```bash
npm start
```
## Структура бази даних

### Проєкт містить 6 таблиць:

* User - Клієнти.

* Restaurant - Заклади.

* MenuItem - Страви.

* Order - Замовлення.

* OrderItem - Позиції в замовленні.

* Courier - Кур'єри.

## Реалізовані сценарії

* Транзакційне замовлення: Створення запису в Order та OrderItems відбувається в одній транзакції з перевіркою цін.

* М'яке видалення: Видалені страви залишаються в БД для історії (поле deletedAt).

* Аналітика: Складний SQL-запит для розрахунку прибутку ресторанів за останній місяць.
